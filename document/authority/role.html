# API

key description use type mark
id 角色 id 主键 String 必填，不可重复
name 角色标识 String 必填，不可重复
title 角色中文名 String 必填
menus 角色菜单 对应菜单 id 字段 Array 可以为空，1 对多
description 角色描述 String
