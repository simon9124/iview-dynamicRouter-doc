# 路由层级

在 <a href="https://github.com/simon9124/iview-dynamicRouter/blob/master/src/mock/role.js" target="_blank">@/mock/role.js</a> 里配置了默认的路由层级如下：

```javascript
const menuLevel = [
  {
    label: '菜单显示该页面选项，页面不含菜单栏',
    value: '1',
  },
  {
    label: '菜单显示该页面选项，页面含菜单栏',
    value: '2',
  },
  {
    label: '菜单隐藏该页面选项，页面不含菜单栏',
    value: '3',
  },
]
```
